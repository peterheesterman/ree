
# Ree

A simplified version of the tree unix utility.

Prints out depth-indented listing of files in a folder (or list of folders).

```
// Installation
cargo install ree

// Usage
ree ./some_folder

// Result
/some_folder
  /some_sub_folder
    file_in_sub_folder.txt
  file_in_folder_1.txt
  file_in_folder_2.txt
```
